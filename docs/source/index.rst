DistFit Pro Documentation
==========================

**DistFit Pro** is a comprehensive Python library for statistical distribution fitting with advanced features:

- 30 statistical distributions (25 continuous, 5 discrete)
- Multiple estimation methods (MLE, Moments, Quantile)
- Goodness-of-Fit tests (KS, AD, Chi-Square, CvM)
- Bootstrap confidence intervals
- Enhanced diagnostics & outlier detection
- Weighted data support
- Multilingual (English, Farsi, German)
